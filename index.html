Introdução

Esta atividade tem como objetivo demonstrar o conhecimento adquirido sobre HTML e CSS, aplicando conceitos de estruturação de documentos web e estilização de páginas.

Desenvolvimento Web

O desenvolvimento web moderno envolve diversas tecnologias e práticas que garantem a criação de sites funcionais, acessíveis e esteticamente agradáveis. As principais tecnologias utilizadas são:

  • HTML (HyperText Markup Language): Linguagem de marcação responsável pela estrutura do documento.
  • CSS (Cascading Style Sheets): Linguagem de estilização que define a apresentação visual dos elementos.
  • JavaScript: Linguagem de programação que adiciona interatividade às páginas web.
